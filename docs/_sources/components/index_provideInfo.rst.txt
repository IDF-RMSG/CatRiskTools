Providing Information to CatRiskTools
====================================================

We welcome the addition of new model listings.

To contribute new model listings to CatRiskTools, or to update existing listings, please contact Stuart Fraser (stuart@disaster-risk.uk), who maintains this for IDF RMSG.


Data can be provided in .xlsx format, using the model dataset templates provided, and lookups for country names and perils names to be used. 
Mandatory fields and optional fields are marked in the template.
